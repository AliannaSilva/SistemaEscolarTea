# Sistema Web Escolar Inclusivo
Sistema web para cadastro e acompanhamento de notas, atividades e observações de alunos, com foco em inclusão de estudantes com TEA.

## Como rodar
1. `npm install`
2. `node src/database/setup.js`
3. `npm run dev`
4. Acesse em `http://localhost:3000`

## Funcionalidades
- Login e autenticação de usuários (professor, coordenador, especialista)
- Cadastro de alunos
- Lançamento de notas e atividades
- Registro de observações comportamentais
- Relatórios de desempenho
